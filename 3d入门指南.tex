我们3D组推荐使用双系统，因为经过我们的测试在虚拟机和WSL2上运行服务器都会很卡顿。

\subsection{什么是双系统？}
简单来说，双系统就像是把你的电脑变成了一栋“双拼别墅”，里面住着两个完全独立的系统（比如 Windows 和 Linux）。
平时它们互不打通、互不干扰。每次你按下电脑开机键的那一秒，屏幕上会出现一个“二选一”的菜单。你点 Windows，电脑就是一台完美的娱乐办公机；你重启电脑点 Linux，它就瞬间变身成一个纯净、高效的开发者专座。
一句话概括：“一机两用，开机选系统，互不包庇，各自拥有 100 \% 的电脑硬件实力。” \textcolor{gray}{(ai生成)}

\subsection{怎么装双系统？}
首先要选择linux发行版，就像你买安卓机还要选择买三星还是小米还是vivo


linux发行版成千上万，可以自行搜索并挑选自己喜欢的发行版，这里介绍几个主流的并且我们团队有使用过的发行版。

\begin{itemize}
    \item Ubuntu: 最流行、生态最好、遇到问题最容易找到答案的发行版。比赛官方的发行版，比赛时服务器和我们的代码都是在Ubuntu上运行的。
    \item Deepin：国产，颜值高，使用简单，内置中文输入法，使用起来类似手机，简单上手。比较推荐。
    \item Popos: 这是我目前一直在使用的系统，它基于Ubuntu，自带现代的桌面环境，（几乎）开箱即用，我很喜欢用它自带的工作区和自动平铺功能。
\end{itemize}

选择好要装的系统后，可以在它官网查看安装方式。

例如deepin的官方安装教程在\url{https://www.deepin.org/zh/deepin-25-installation/}

不同发行版的安装方式大同小异。注意提前备份电脑里重要的文件，以免误操作导致重要内容丢失（例如装popos的时候会让你选全盘安装还是双系统，如果选全盘安装，那电脑里的东西全都没了。）

\section{学习使用git}
\subsection{git有什么用？}

有了git以后，你再也不用担心把代码改坏了，你可以像游戏一样拥有存档点，可以随时回到任意一个存档。搭配github，你可以把你的代码上传到远程仓库，实现队伍里的协同合作。

git不仅可以创建存档，还可以创建平行世界，你在你的平行世界里修改代码不会影响你队友的代码，当代码确定无误了之后，可以把你的平行世界里的代码同步到主世界里。

刚开始学习git可能会觉得很难，很不理解，这是正常的，上手了之后就离不开它了，这里推荐一个工具lazygit，可以大大降低上手难度。

\section{开始我们的项目}
终于可以开始我们的项目了


但是在开始之前，我们要先安装两个工具: pipx 和 uv

\begin{itemize}
    \item pipx是一个专门用来管理和安装python命令行应用的工具，安装命令如下。
\end{itemize}

\begin{lstlisting}[language=bash]
sudo apt install pipx
\end{lstlisting}

\begin{figure}[htbp]
    \centering % 让图片居中
    % 插入图片，宽度设为页面文本宽度的 80% (0.8)
    \includegraphics[width=1\textwidth]{pipx.png} 
    
    \caption{pipx --version可以用来验证是否安装成功(uv同理)} % 图片底部的标题
\end{figure}

\begin{itemize}
    \item uv是目前 Python 领域最颠覆性的“全家桶”级包管理工具，我们用它来创建和管理虚拟环境(什么是虚拟环境？简单来说，虚拟环境就是一个独立、隔离的“沙盒”文件夹。它允许你在同一台电脑上，为不同的项目安装不同版本、互不干扰的 Python 包和依赖。) 安装命令如下。
\end{itemize}

\begin{lstlisting}[language=bash]
curl -LsSf https://astral.sh/uv/install.sh | sh
\end{lstlisting}

接下来开始安装服务器和底层代码

\begin{itemize}
    \item 服务器: \url{https://gitlab.com/robocup-sim/rcssservermj}
    \item 底层代码: \url{https://gitlab.com/bahiart/BahiaRT-MujOCo-base}
\end{itemize}

安装服务器命令如下
\begin{lstlisting}[language=bash]
pipx install rcssservermj
\end{lstlisting}

启动服务器的命令是
\begin{lstlisting}[language=bash]
rcssservermj
\end{lstlisting}


\bigbreak
拉取底层代码的命令为
\begin{lstlisting}[language=bash]
git clone https://gitlab.com/bahiart/BahiaRT-MujOCo-base.git
\end{lstlisting}

\bigbreak
然后我们需要为底层代码安装依赖
\smallbreak
首先进入BahiaRT-MujOCo-base文件夹里，确保当前路径下有pyproject.toml这个文件，使用以下命令即可自动安装所有依赖
\begin{lstlisting}[language=bash]
uv sync
\end{lstlisting}

等待依赖安装完成后，确保服务器已经启动，在当前路径下运行以下命令即可启动底层代码
\begin{lstlisting}[language=bash]
uv run ./start.sh
\end{lstlisting}
